Instagram Network Analysis

This project explores the structure of my Instagram social network by modeling mutual connections as a graph. Using tools like Playwright and Instaloader, I scraped my follow/follower data and constructed a directed graph representing who follows whom. I then used NetworkX for analysis and PyVis for interactive visualization.

Highlights:

  • Scraped my Instagram network using automated browser tools and APIs
  • Built a directed graph of user relationships using networkx
  • Applied community detection using the Louvain method
  • Calculated centrality metrics: in-degree, out-degree, clustering coefficient, etc.
  • Visualized the graph with color-coded communities using PyVis

View the Interactive Graph

Reflection

This project taught me how social connections cluster into communities—and how some nodes (people!) play bridging roles between them. It also showed me how useful graph theory can be in modeling real-world data, and the importance of scalable tooling when working with platform APIs.

You can find the full source code on GitHub, or read more about my data visualization and ML work in the projects section of my site.
